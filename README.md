# Jibril Portfolio Website

个人作品集网站，使用现代前端技术栈构建。

## 技术栈

- HTML5
- Tailwind CSS
- PostCSS
- Autoprefixer

## 项目结构

```
jibril-website/
├── src/
│   ├── assets/      # 静态资源
│   ├── components/  # 组件
│   ├── styles/      # 样式文件
│   └── index.html   # 主页面
├── dist/            # 构建输出目录
├── node_modules/    # 依赖包
└── package.json     # 项目配置
```

## 安装

1. 克隆项目
```bash
git clone [项目地址]
cd jibril-website
```

2. 安装依赖
```bash
npm install
```

## 开发

运行开发服务器：
```bash
npm run dev
```

## 构建

构建生产版本：
```bash
npm run build
```
